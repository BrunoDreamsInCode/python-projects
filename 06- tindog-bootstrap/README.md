# 🐶 TinDog - Bootstrap Practice Project

This project is a simple landing page built as a practice exercise using **Bootstrap 5**.

The goal was not to create a fully functional product, but to improve my skills with layout, spacing, and responsive design using Bootstrap utilities.

---

## 🚀 About the Project

**TinDog** is a fictional app inspired by Tinder — but for dogs 🐕.

This project focuses on:

* Using Bootstrap grid system
* Practicing spacing (padding, margin)
* Working with responsive layouts
* Structuring sections like a real landing page
* Combining Bootstrap with custom CSS

⚠️ Note:
This is a **non-functional UI project**. Buttons, links, and footer elements do not have real actions.

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* Bootstrap 5 (via CDN)

---

## 📱 Features

* Responsive layout using Bootstrap grid
* Hero section with CTA buttons
* Features section with icons
* Testimonial section
* Pricing cards with aligned buttons
* Footer (static, for layout purposes only)

---

## 🎯 What I Practiced

* `container`, `row`, `col` structure
* Flexbox utilities (`d-flex`, `flex-column`, `mt-auto`)
* Buttons and card components
* Responsive breakpoints (`col-lg`, `col-md`, etc.)
* Alignment and spacing adjustments
* Reusable UI sections
